## 📊 Key Steps Performed

1. **Data Loading**: Used `train.csv` from Kaggle dataset.
2. **Feature Selection**:
   - `GrLivArea` (square footage)
   - `BedroomAbvGr` (number of bedrooms)
   - `FullBath`, `HalfBath` (bathroom counts)
3. **Train-Test Split**: 80% training / 20% testing
4. **Model Training**: `LinearRegression()` from scikit-learn
5. **Evaluation**:
   - R² Score
   - Mean Squared Error (MSE)
6. **Visualization**: Plotted Actual vs Predicted prices
